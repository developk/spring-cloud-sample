## 프로젝트 셋팅 및 어플리케이션 실행

*nodejs가 설치되어 있어야 합니다.*

```
cd mes-admin-web // mes-admin-web 경로로 이동

npm install // package.json에 기술 된 의존 라이브러리 설치

npm run serve // live reload 상태의 개발용 서버 실횅.

npm run build // production 용 build
```
